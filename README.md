# Zaden Maestas — Portfolio

Personal portfolio website built with [Astro](https://astro.build/).

## Stack

- **Framework:** Astro 6
- **Icons:** [astro-icon](https://github.com/natemoo-re/astro-icon) + [Lucide](https://lucide.dev/) — SVGs sourced from [SVG Repo](https://www.svgrepo.com/)
- **Font:** Lato (Google Fonts)
- **Package manager:** pnpm

## Getting Started

**Prerequisites:** Node.js ≥ 22.12.0

```bash
pnpm install
pnpm dev
```

| Command        | Action                          |
|----------------|---------------------------------|
| `pnpm dev`     | Start dev server at `localhost:4321` |
| `pnpm build`   | Build for production to `./dist/` |
| `pnpm preview` | Preview the production build    |
